Enchantavator

Enchantavator is a 2D rouge-like survival game that rank in the top 20 for the Zeno Game Jam. The theme of the game jam was "going up." As the solo programmer, I'm really proud of this game because I had to figure out and make every single part of code myself. Many of the challenges of this game were due to the time crunch of having one week. This game taught me how to create randomized enemy spawns, create a rouge-like shop menu, and how to fluidly change between game states.
